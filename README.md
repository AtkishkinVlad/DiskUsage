## Description

This is a CLI tool for disk space analysis

## What you need to start

* Python 3.7 or newest
* [tqdm library](https://pypi.org/project/tqdm/)

## Launch Options

**path => required parameter**

============================================================

* `-h` - usage guide
* `-s` - sort options (name, size, modify time)
* `-r` - reverse sort order
* `-d` - scan depth
* `-n` - scan without progress bar
* `-t` - top element by size
* `-b` - files that take up so many percent of the space
* `-e` - print { file extension: size }
