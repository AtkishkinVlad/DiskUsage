Утилита du

Пример запуска -> du.py [путь] -sort s(или l)(сортировка по размеру или алфавиту) -depth 2(вложенность)

Функции

convert_bytes -> дает удобное представление размера файлов
get_dir_size(path) -> дает размер нашей папки (с подсчетом размеров подпапки)
get_dirs(path, max_depth) -> список папок в директории
get_dirs_with_size(dirs) -> список директорий с их размером
sort_by_size(dirs) -> сортирует по возрастанию
sort_by_lexical(dirs) -> сортирует по названию
